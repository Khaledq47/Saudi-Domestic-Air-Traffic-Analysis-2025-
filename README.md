#  Saudi Domestic Air Traffic Analysis (2025)

##  Overview

This project analyzes domestic air traffic in Saudi Arabia for the year 2025 using Power BI.
The goal is to uncover patterns in passenger movement, flight activity, and airport performance, and to transform raw data into meaningful business insights.

##  Objectives

* Analyze monthly trends in flights and passengers
* Identify the busiest airports and most popular destinations
* Compare arrival and departure operations
* Evaluate route efficiency using passenger-to-flight ratios

---

##  Dataset

The dataset includes domestic flight records with the following attributes:

* Year and Month
* Airport (origin)
* Destination
* Arrival / Departure indicator
* Number of Flights
* Number of Passengers

---

##  Data Preparation

Data cleaning and transformation steps included:

* Handling missing destination values
* Ensuring consistent formatting for categorical fields
* Converting data types (numeric and text)
* Creating calculated columns for time-based analysis

---

##  Key Metrics (KPIs)

* Total Flights
* Total Passengers
* Average Passengers per Flight
* Number of Airports
* Number of Destinations

---

##  Dashboard Features

The Power BI dashboard includes:

* **Monthly Trends:** Line charts for flights and passengers
* **Top Airports:** Passenger volume by airport
* **Top Destinations:** Most traveled routes
* **Arrival vs Departure:** Operational comparison

---

##  Key Insights

* Passenger traffic shows clear variation across months, indicating seasonal demand patterns
* A small number of airports handle the majority of traffic, acting as key hubs
* Flight activity is concentrated on specific destinations, suggesting strong route demand
* Arrival and departure volumes are generally balanced, reflecting stable operations
* Some routes achieve higher passenger efficiency, indicating better utilization of flights
